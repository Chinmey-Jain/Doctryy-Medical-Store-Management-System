Doctryy

Online Pharmacy Management System


This project has the implementation of HTML,CSS & Javascript.
The backend has a Relational Database made with MySQL in Xampp.
The connection between the database and the website is established using php.
The programming for the front-end part was done using Visual Studio Code and testing was done use the Live Server extension.
It is not nessesary to use the same IDE for development, it is solely the user's choice.
Also some parameters were established both in the database and in the JS code itself to check data like passwords, etc.
All the attachment files were kept in the same root directory.

This is more or less all the major highlights of my project.
Hope it helps!!!
